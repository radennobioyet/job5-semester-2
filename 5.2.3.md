jawaban
1. Pada faktorialDC, penggunaan blok if (n == 1) berfungsi sebagai base case (titik berhenti rekursi) yang akan langsung mengembalikan nilai 1 ketika proses pemecahan masalah sudah mencapai elemen paling kecil. Sedangkan blok else adalah tahapan divide dan conquer, di mana fungsi akan memanggil dirinya sendiri lagi namun dengan nilai parameter yang terus mengecil (n-1) hingga menyentuh base case. 
2. Sangat memungkinkan. Perulangan pada metode iteratif faktorialBF() bisa diubah menggunakan while ataupun do-while

```
int faktorialBF(int n) {
    int fakto = 1;
    int i = 1; // 1. Inisialisasi dipindah ke luar perulangan

    while (i <= n) { // 2. Kondisi batas tetap dievaluasi
        fakto = fakto * i;
        i++; // 3. Increment dilakukan di dalam blok perulangan
    }

    return fakto;
}
```

3. Statement fakto *= i; adalah proses penggabungan/kalkulasi untuk algoritma Brute Force yang dilakukan secara iteratif (berulang-ulang secara linier) dalam sebuah looping. Sebaliknya, int fakto = n * faktorialDC(n-1); adalah implementasi proses secara rekursif, di mana perhitungan dilakukan dengan cara memanggil metodenya sendiri untuk mendapatkan hasil perkalian dari nilai n saat ini dan hasil faktorial n sebelumnya. 

4. Kesimpulan: Method faktorialBF() bekerja secara langsung dan lurus dengan mengalikan deret angka satu per satu secara iteratif dari awal hingga mencapai iterasi n. Di sisi lain, faktorialDC() bekerja dengan membagi masalah secara terus menerus (pemanggilan rekursif n-1) hingga mentok di angka paling dasar (n=1), lalu setelah mencapai dasar ia baru akan menggabungkan/mengalikan total kalkulasinya (combine).
