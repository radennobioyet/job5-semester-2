Pertanyaan
1. Kenapa dibutuhkan variable mid pada method TotalDC()? 
2. Untuk apakah statement di bawah ini dilakukan dalam TotalDC()?
```
        double lsum = totalDC(arr, l, mid);
        double rsum = totalDC(arr, mid + 1, r);
```
3. Kenapa diperlukan penjumlahan hasil lsum dan rsum seperti di bawah ini?

```
 return lsum + rsum;
```
4. Apakah base case dari totalDC()?
5. Tarik Kesimpulan tentang cara kerja totalDC()





Jawaban
1. Variabel mid sangat dibutuhkan untuk menentukan indeks penanda titik tengah (median). Titik tengah ini berfungsi untuk membelah (tahap divide) suatu array menjadi tepat dua bagian (sub-array kiri dan sub-array kanan) yang akan diproses pada tahap selanjutnya. 

2. Statement double lsum = totalDC(arr, l, mid); dan double rsum = totalDC(arr, mid+1, r); adalah tahap conquer. Pemanggilan rekursif ini dilakukan untuk mencari perhitungan jumlah total nilai dari elemen-elemen array bagian paling kiri (l sampai mid) dan menghitung total nilai array dari pecahan bagian kanan (mid+1 sampai r). 

3. Proses penjumlahan lsum dan rsum (return lsum+rsum;) merupakan tahap akhir (Combine). Karena pada tahap sebelumnya algoritma memecah perhitungan dalam blok array sebelah kiri dan sebelah kanan, maka diperlukan penjumlahan ini untuk menyatukan dan mendapatkan total kalkulasi keseluruhan. 

4. Base case (syarat berhenti) dari totalDC() adalah saat kondisi indeks left telah bernilai sama persis dengan right, yaitu if (l == r). Kondisi ini menandakan bahwa pecahan array saat ini hanya tersisa 1 elemen saja (sudah tidak bisa dibagi 2 lagi). 

5. Kesimpulan: totalDC() akan berjalan dengan membelah keseluruhan array yang berisi sejumlah nilai (keuntungan) menjadi dua sub-array yang lebih kecil secara berulang-ulang hingga hanya berisikan masing-masing 1 elemen (divide). Selanjutnya, program menjumlahkan pasangan 1 elemen tersebut mulai dari pecahan yang paling kecil, disusul hasil penjumlahannya dikumpulkan dan dijumlahkan ke tingkat atas secara terus menerus (conquer & combine) untuk mendapatkan hasil nilai penjumlahan array paling besar/lengkap.
